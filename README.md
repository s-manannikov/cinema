## cinema

[![Build Status](https://travis-ci.org/s-manannikov/cinema.svg?branch=master)](https://travis-ci.org/s-manannikov/cinema)

This project is a web application for cinema tickets booking.

Technologies
- Java Servlet API
- Apache Tomcat
- PostgreSQL
- JS (jQuery, AJAX), HTML, CSS
- Maven
